# Revibe AI Demo Script (3-5 Minutes)

## 1. Opening (20-30 sec)

1. "Revibe AI helps users turn e-waste into practical, beginner-friendly reuse projects."
2. "The product combines AI recommendations, nearby resource discovery, community sharing, and GitHub identity."

## 2. Upload and Analyze (60-90 sec)

1. Open `/upload`.
2. Add an image and click **Analyze**.
3. Explain that backend AI returns structured JSON:
   - material
   - confidence
   - risk
   - sustainability score
   - practical ideas
   - actionable steps

## 3. Results and Resources (60-90 sec)

1. On `/results`, highlight safety-aware recommendations.
2. Scroll to **Nearby Resources**.
3. Click list items and show marker-linked map behavior.
4. Clarify MVP scope: currently curated sample resource points.

## 4. Community Flow (40-60 sec)

1. Open `/community`.
2. Create a quick progress post.
3. Show feed update and explain collaborative learning angle.

## 5. GitHub Integration (40-60 sec)

1. Open `/profile`.
2. Click **Connect GitHub** (OAuth).
3. Show connected profile and featured repositories.
4. Explain this links public developer identity to sustainability work.

## 6. Closing (20-30 sec)

1. "Revibe AI demonstrates an end-to-end MVP: analysis -> action -> visibility."
2. "It is deployment-ready with clear frontend/backend separation and secure env handling."
